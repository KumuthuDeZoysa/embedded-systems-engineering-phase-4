```

```

### **Why you see this:**

```mermaid
Device                        Cloud Server
  |                                |
  |------ GET /api/config -------->| 
  |       (NO HMAC!)               |
  |                                |
  |                           [Check HMAC]
  |                           ‚ùå Missing!
  |                                |
  |<------ 401 Unauthorized -------|
  |                                |
```

**This is CORRECT behavior!** 

- ‚úÖ Cloud has security enabled
- ‚úÖ Cloud expects HMAC authentication
- ‚ùå Device sends plain HTTP (no HMAC)
- ‚úÖ Cloud rejects ‚Üí 401 error

**After integration, it will be:**

```mermaid

  |--- Secured Message ----------->|
```
